# 🚀 Sales Management & Analytics Dashboard

A full-stack sales management system with an integrated analytics dashboard, built to streamline business operations and visualize performance in real-time.

---

## 🧩 Features

### 👤 User
- Register & Login (JWT Authentication)
- Browse products
- Create orders

### 🛠 Admin
- Manage products, customers, employees
- View analytics dashboard:
  - Monthly revenue
  - Top-selling products
  - Top customers

---

## 🛠 Tech Stack

### Frontend
- React
- Axios
- React Router
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- SQL Server
- Stored Procedures
- Triggers
- Functions

---

## 📊 Database Highlights

- Designed relational database with foreign keys and constraints  
- Implemented triggers for:
  - Updating product quantity after order
- Used stored procedures for:
  - Creating orders
  - Fetching analytics data  
- Created functions for:
  - Calculating total invoice value

## ⚙️ Installation & Setup
## 🚀 Quick Start

Follow the steps below to run this project locally:
1. Setup Backend
   cd server
   npm install
   npm start
2. Setup Frontend
   cd client
   npm install
   npm start
3. Setup Database
   Open SQL Server
   Run scripts in /database folder

### 2. Clone repository
```bash
git clone https://github.com/your-username/sales-management-dashboard.git
'''
